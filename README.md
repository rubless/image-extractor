# 📂 이미지 추출기 (Image Extractor)

여러 폴더에 흩어져 있는 이미지를 한 곳으로 모아주는 도구입니다.

---

## ✨ 주요 기능

### 🖱️ 드래그 앤 드롭
- 폴더를 창에 끌어다 놓으면 자동으로 이미지를 탐색합니다.
- 하위 폴더가 있으면 **최하위 폴더까지** 모두 탐색합니다.

### 📁 저장 경로 지정
- 추출한 이미지를 저장할 폴더를 직접 선택할 수 있습니다.
- **매 작업마다 새로운 경로를 지정**할 수 있어 유연하게 사용할 수 있습니다.

### 🔢 수량 제한
- 각 폴더에서 꺼내올 이미지 수를 설정할 수 있습니다. (1~9999)
- 꺼내오는 기준은 **파일 수정 시간이 최신인 순서**입니다.
- 「전체 추출」 토글을 켜면 수량 제한 없이 폴더 내 **모든 이미지**를 추출합니다.

### 🔒 원본 보존
- 원본 파일은 건드리지 않고, **복사** 방식으로 가져옵니다.
- 파일명이 겹칠 경우 자동으로 번호를 붙여 저장하므로 덮어쓸 걱정이 없습니다.

---

## 🖼️ 지원 형식

`jpg` `jpeg` `png` `bmp` `gif` `tiff` `webp`

---

## 💻 사용 방법

1. 프로그램을 실행합니다.
2. **「저장 경로 선택」** 버튼을 눌러 이미지를 모을 폴더를 지정합니다.
3. 꺼내올 이미지 수량을 설정하거나, 전체 추출 토글을 켭니다.
4. 이미지가 들어있는 **폴더를 창에 드래그 앤 드롭**합니다.
5. 추출 결과가 로그에 표시됩니다.
